# System Performance Dashboard

A lightweight system performance dashboard built with Python (Flask) and `psutil`.
It monitors CPU, RAM, Disk, and Network activity in real-time.

## Installation

```bash
pip install -r requirements.txt
```

## Running the App

```bash
python app.py
```

The app listens on `0.0.0.0:8080`. Access it via `http://<server-ip>:8080`.
